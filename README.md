# Project Title

Code base for Jarry Chen's Assignment One in Prog of Internet of Things course at RMIT (2018). This github will contain 3 projects that run python3 as its programming language. This code is meant to be run on a raspberry pi with a sensehat installed 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Pull the project using https://github.com/JayJayc/Prog_IoT_Assignment_1.git

Using a SFTP service import the code into raspberry enviroment

Run the following files with python3:
* create_db.py
* create_name_db.py
* cron.py

example:
```
python3 create_db.py
```

Run save_name.py to register the phone with the raspberry pi

To start the web server:
Use the command : sudo python3 ~/YourPATH/WebService/webserver.py

### Prerequisites

* Python3
* Raspberry Pi 3 Model B
* Sense hat
